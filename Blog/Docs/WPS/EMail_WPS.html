Lets Observe the Property of an MailMessage object.
Property Description
From Sender Mail Address
To Gets the address collection that contains the recipients of this e-mail message.
Subject Gets or sets the subject line for this e-mail message.
Attachment Gets the attachment collection used to store data attached to this e-mail message.
Header Gets the e-mail headers that are transmitted with this e-mail message.
Priority Gets or sets the priority of this e-mail message.
DiliveryNotificationOptions Gets or sets the delivery notifications for this e-mail message.
BodyEncoding Gets or sets the encoding used to encode the message body.
BCC Gets the address collection that contains the blind carbon copy (BCC) recipients for this e-mail message.
AlternateViews Gets the attachment collection used to store alternate forms of the message body.

SMTPClient Setup

Allows applications to send e-mail by using the Simple Mail Transfer Protocol (SMTP).

To define a SMTPClient Object We Use “System.Net.Mail.SMTPClient” Class.


Constructor
Name Description
SmtpClient() Initializes a new instance of the SmtpClient class by using configuration file settings.
SmtpClient(string) Initializes a new instance of the SmtpClient class that sends e-mail by using the specified SMTP server.
SmtpClient(String, Int32) Initializes a new instance of the SmtpClient class that sends e-mail by using the specified SMTP server and port.
Property Description
Host Gets or sets the name or IP address of the host used for SMTP transactions.
Port Gets or sets the port used for SMTP transactions.
EnableSsl Specify whether the SmtpClient uses Secure Sockets Layer (SSL) to encrypt the connection.
UseDefaultCredentials Gets or sets a Boolean value that controls whether the DefaultCredentials are sent with requests.
ClientCertificates Specify which certificates should be used to establish the Secure Sockets Layer (SSL) connection.
Credentials Gets or sets the credentials used to authenticate the sender.

Using SmtpClient Object we can set these above properties.
