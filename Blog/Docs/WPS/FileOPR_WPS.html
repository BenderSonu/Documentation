Reading Entries with WPS

To get input from command line we use the following cmdlet. 

Read-Host

The Read-Host cmdlet enables you to interactively prompt a user for information. 

Example$Name = Read-Host "Please enter your name"Description: This is Prompt user for his/her Name and store into variable named “Name”.

NOTE: Colon is not required at the end of line. Windows PowerShell Adds colon itself. 

OPTIONSRead-Host -AsSecureString "Enter Password"Used to mask data entered at prompt.

Writing Entries with WPS

To get output on command line we use the following cmdlet. 

Write-Host

The Write-Host cmdlet enables you to write messages to the Windows PowerShell console.


Lets Take an Example 

Example$Name = Write-Host "Please enter your name"

Options Description
ForegroundColor Enables to specify different Text Color.
BackgroundColor Enables to specify Text Background Color.
NoNewLine Lets your cursor to stay on the current line
